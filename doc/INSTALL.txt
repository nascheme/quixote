Installing Quixote
==================

Quixote requires Python 3.13 or later.  The project is packaged with
``pyproject.toml``; there is no ``setup.py``.

To install into the current environment::

    python3 -m pip install .

Development is managed with ``uv``.  To set up a checkout for development
(this creates a virtual environment and installs Quixote plus its dev
dependencies)::

    uv sync

and run tools through ``uv run`` (for example ``uv run pytest``).

If you have an older Quixote installed and want to find where, run::

    python3 -c "import os,quixote;print(os.path.dirname(quixote.__file__))"

(If the import fails, you have no existing installation.)


Quick start
===========

In a terminal window, run::

    python3 -m quixote run

In a browser, open http://localhost:8080
